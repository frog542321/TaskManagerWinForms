## Описание проекта

TaskManager — это приложение Windows Forms для управления задачами.

Программа позволяет:

- Добавлять задачи
- Редактировать задачи
- Удалять задачи
- Отмечать задачи как выполненные
- Фильтровать задачи по приоритету
- Фильтровать выполненные/невыполненные задачи
- Отображать просроченные задачи красным цветом
- Отображать актуальные задачи зеленым цветом
